Below is the complete curriculum for our flagship Data Analytics Beginner Certification

Module 1: Data Analytics Fundamentals

+
  • Introduction to data analytics 2 hours
  • Data types and structures 2 hours
  • Analytics process overview 2 hours
  • Case study: Retail sales analysis 2 hours
  • Assignment: Define analytics goals 2 hours

Module 2: Data Cleaning with Excel

+
  • Excel for data cleaning 2 hours
  • Handling missing data 2 hours
  • Data formatting techniques 2 hours
  • Case study: Cleaning customer data 2 hours
  • Assignment: Clean a dataset 2 hours

Module 3: SQL for Data Analytics

+
  • Introduction to SQL 2 hours
  • Basic SQL queries 2 hours

Module 4: Data Visualization

+
  • Introduction to Tableau 2 hours
  • Creating charts and graphs 2 hours
  • Building dashboards 2 hours
  • Case study: Visualizing sales data 2 hours
  • Assignment: Create a dashboard 2 hours

Module 5: Descriptive Statistics

+
  • Measures of central tendency 2 hours

  • Case study: Customer behavior analysis 2 hours
